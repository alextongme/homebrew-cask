# homebrew-cask

Homebrew Cask formulas by [@alextongme](https://github.com/alextongme).

## Install

```bash
brew tap alextongme/cask
brew install --cask count-tongulas-eye-break
```

## Available Casks

| Cask | Description |
|------|-------------|
| `count-tongulas-eye-break` | Menu bar app that reminds you to rest your eyes using the 20-20-20 rule |
